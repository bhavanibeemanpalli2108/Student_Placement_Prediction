## **Data Dictionary**

* student_id - Unique identifier for each student
* age - Student age
* gender - Gender of the student
* cgpa - Cumulative Grade Point Average
* branch - Academic branch/department
* college_tier - College tier classification
* internships_count - Number of internships completed
* projects_count - Number of projects completed
* certifications_count - Number of certifications earned
* coding_skill_score - Coding skill assessment score
* aptitude_score - Aptitude test score
* communication_skill_score - Communication skill score
* logical_reasoning_score - Logical reasoning score
* hackathons_participated - Number of hackathons participated in
* github_repos - Number of GitHub repositories maintained
* linkedin_connections - Number of LinkedIn connections
* mock_interview_score - Mock interview performance score
* attendance_percentage - Student attendance percentage
* backlogs - Number of academic backlogs
* extracurricular_score - Extracurricular activity score
* leadership_score - Leadership assessment score
* volunteer_experience - Indicates volunteer experience (Yes/No)
* sleep_hours - Average sleep hours per day
* study_hours_per_day - Average study hours per day
* placement_status - Placement outcome (Placed/Not Placed)
* salary_package_lpa - Salary package offered in LPA (Lakhs Per Annum)
